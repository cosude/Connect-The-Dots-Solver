# Connect the Dots Solver Simulation
A real-time visualization of well-known searching and csp algorithms on the "Connect the Dots" puzzle using Pygame.

## Algorithms Implemented
* **Search-Based:** A*, BFS, DFS, UCS, IDA*
* **Constraint-Based:** CSP Solver with Forward Checking and Backtracking

## Key features
* **Original Heurisitc algorithm** 
* **Real-time Race Mode**

## Installation
1. Clone the repo: `git clone https://github.com/cosude/Connect-The-Dots-Solver.git`
2. Install dependencies: `pip install pygame colorama`
3. Run the app: `python main.py`
